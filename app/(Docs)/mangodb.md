🧠 What is MongoDB?
MongoDB is a NoSQL database used to store data in a flexible, JSON-like format.

JSON: 
{
  "name": "Rahul",
  "age": 20
}   


1. Database
A container of collections 
Like a folder

2. Collection
Group of documents
Like a table in SQL

3. Document
A single record (JSON-like)



{
  "name": "Aman",
  "age": 18,
  "skills": ["JavaScript", "MongoDB"]
}
